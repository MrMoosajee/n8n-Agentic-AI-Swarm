## ğŸ“ KEY DESIGN PRINCIPLES

1. **Zero-Cost First**
   - Every component must be free or self-hosted
   - No monthly subscriptions or usage fees
   - Hardware you already own doesn't count

2. **Cloud for Reasoning, Local for Privacy**
   - Use cloud AI for complex decision-making
   - Keep code generation local (no leaking IP)
   - Hybrid approach maximizes both

3. **Version-Agnostic Integration**
   - Use generic HTTP APIs over integrations
   - Avoid version-locked node types
   - Future-proof against updates

4. **Fail-Safe State Management**
   - All progress saved to Postgres
   - Crash-resistant (can resume anywhere)
   - No in-memory-only state

5. **Observable & Debuggable**
   - Every agent action logged
   - Full audit trail in database
   - Easy to trace failures
